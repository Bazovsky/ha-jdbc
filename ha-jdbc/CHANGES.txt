HA-JDBC Versions
================

1.0-alpha-3
-----------
* Upgraded JGroups to v2.2.6
* Fixed multi-threading bug in SQLProxy
* Changed license to LGPL - added license details to source code
* Removed obsolete RowSetProxy
* Refactored package names to reflect project name

1.0-alpha-2
-----------
* Added ability specify url of configuration file using "ha-jdbc.configuration" system property
* Extended classpath search for configuration file to include system classpath
* Fixed DataSource proxy initialization
* DriverProxy now expects the cluster name to match the JDBC url
* Renamed DistributedDatabaseClusterManager -> DistributableDatabaseClusterManager
* Renamed pooled-datasource -> pool-datasource in configuration DTD and JiBX mapping
* Fixed bug in clean target of ant build file

1.0-alpha-1
-----------
* Initial release
